Introduction

PRESCIENT is a tool for simulating cellular differentiation trajectories with arbitrary cell state intializations.

Installation

PyPI Installation

We recommend using pip to install PRESCIENT.
pip install prescient

Github Installation

git clone git@github.com:gifford-lab/prescient.git cd prescient pip install .
